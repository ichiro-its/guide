Remote Desktop Using VNC
========================

`Virtual Network Computing <https://en.wikipedia.org/wiki/Virtual_Network_Computing>`_ (VNC) is a graphical desktop-sharing system that uses the Remote Frame Buffer protocol (RFB) to remotely control another computer.
It transmits the keyboard and mouse input from one computer to another, relaying the graphical-screen updates, over a network.

This section will explain a guide on how to remote desktop a machine using VNC.
